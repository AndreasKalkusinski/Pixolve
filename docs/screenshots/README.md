# Screenshots Guide

This directory contains screenshots for the main README.md file.

## Required Screenshots

Please add the following screenshots to this directory:

1. **main-interface-light.png** - Main interface in light theme
   - Show: Complete window with loaded images, settings visible
   - Resolution: 1920x1080 or similar
   - Theme: Light mode

2. **main-interface-dark.png** - Main interface in dark theme
   - Show: Same view as above but in dark mode
   - Resolution: 1920x1080 or similar
   - Theme: Dark mode

3. **multi-format-export.png** - Multi-format export feature
   - Show: Multi-Format Export checkbox enabled with format options visible
   - Highlight: The horizontal layout of format checkboxes with quality settings
   - Theme: Either light or dark

4. **batch-conversion.png** - Conversion in progress
   - Show: Progress bar active, multiple images with "Converting" status
   - Show: Real-time statistics in the status bar
   - Theme: Either light or dark

5. **per-image-settings.png** - Custom settings flyout
   - Show: Gear icon clicked with settings flyout open
   - Show: Format, Quality, and Max Pixel custom fields
   - Theme: Either light or dark

## Screenshot Tips

- Use Windows Snipping Tool (Win+Shift+S) or similar tool
- Capture the full application window including shadow
- Use realistic sample images (nature, photos, graphics)
- Show meaningful file names and realistic file sizes
- Ensure UI text is crisp and readable
- PNG format with good compression

## How to Take Screenshots

1. Launch Pixolve
2. Load sample images from `C:\Users\<YourUser>\Pictures` or similar
3. Set up the UI for each screenshot scenario
4. Capture using Snipping Tool
5. Save with the exact filename listed above
6. Verify the screenshot appears correctly in README.md

## Verifying

After adding screenshots, preview README.md in GitHub or VS Code to ensure:
- Images load correctly
- File paths are correct
- Images are appropriately sized
- Visual quality is good
